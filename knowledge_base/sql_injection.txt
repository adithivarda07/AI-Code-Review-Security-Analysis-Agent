SQL Injection

Definition
SQL Injection is a vulnerability where user input modifies SQL queries.

Example

Bad:
query = "SELECT * FROM users WHERE id=" + user_id

Good:
cursor.execute(
    "SELECT * FROM users WHERE id=?",
    (user_id,)
)

Risks
- Database compromise
- Authentication bypass
- Data leakage
- Remote code execution

Prevention
- Prepared Statements
- Parameterized Queries
- ORM Frameworks
- Input Validation

Severity
High
