Hardcoded Password

Definition
Hardcoded passwords are credentials written directly inside the source code.

Why is it dangerous?
If attackers gain access to the source code, they immediately obtain valid credentials.

Risks
- Unauthorized access
- Data breaches
- Credential leakage
- Compliance violations

Best Practices
- Store passwords using environment variables.
- Use secret managers.
- Never expose credentials in version control.

Example

Bad:
password = "admin123"

Good:
import os
password = os.getenv("DB_PASSWORD")

Severity
High
